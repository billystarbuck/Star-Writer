Write without a meter running

Star Writer uses the included Qwen2.5-14B model as its local writing engine. Once installed, the built-in model can run from your computer without monthly AI fees, lifetime fees, subscriptions, or paid token usage. Use it as much as your computer can handle.

You can also connect a supported outside AI provider with your own API credentials. Outside providers may charge their own usage fees. Those charges are separate from Star Writer and are not required to use the included local model.

Built for serious writing projects

  • Plan worlds, characters, outlines, chapters, and complete manuscripts.
  • Keep separate Story Memory files so the AI can follow the correct project.
  • Revise, evaluate, check continuity, and work with reusable prose profiles and templates.
  • Organize chapters and export writing to Word, PDF, and HTML.
  • Work locally with computer and file permissions turned off by default until you choose to enable them.

Before you download

Star Writer is beta software. Back up important writing before testing. Windows may display a SmartScreen warning because this is a new independent application. Confirm that the file came from this page and that the publisher reads Billy Starbuck Star Writer Code Signing.

System requirements

  • 64-bit Windows 10 version 1809 or later, or Windows 11
  • 16 GB RAM minimum; 24 GB or more recommended
  • Approximately 40 GB of free disk space during download and installation
  • A capable modern processor; a supported GPU and additional RAM can improve generation speed
